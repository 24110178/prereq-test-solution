  1. How many multiplications and additions do you need to perform a matrix multiplication between a (n, k) and (k, m) matrix? Explain.

To multiply a matrix A of shape $(n, k)$ and a matrix B of shape $(k, m)$, we produce a resulting matrix C of shape $(n, m)$.

Let's consider how a single element $C_{ij}$ (the element in the $i$-th row and $j$-th column) of the resulting matrix is calculated. It's the dot product of the $i$-th row of A and the $j$-th column of B.

$$ C_{ij} = \sum_{p=1}^{k} A_{ip} \times B_{pj} $$

To calculate this single element $C_{ij}$:

  • Multiplications: We perform $k$ multiplications (one for each term $A_{ip} \times B_{pj}$).
  • Additions: We perform $k-1$ additions to sum the $k$ products.

Since the resulting matrix C has $n \times m$ elements, we can find the total number of operations:

  • Total Multiplications: $(n \times m) \times k = nmk$
  • Total Additions: $(n \times m) \times (k-1) = nm(k-1)$

So, you need $nmk$ multiplications and $nm(k-1)$ additions to perform the matrix multiplication.

def multiply_list_of_lists(A, B):
    
    n, k1 = len(A), len(A[0])
    k2, m = len(B), len(B[0])

    if k1 != k2:
        raise ValueError("Incompatible matrix dimensions for multiplication")

    C = [[0 for _ in range(m)] for _ in range(n)]  # result matrix with zeros

    for i in range(n):   # multiplication
        for j in range(m):
            for p in range(k1):
                C[i][j] += A[i][p] * B[p][j]
    return C

  1. Finding the highest element in a list requires one pass of the array. Finding the second highest element requires 2 passes of the array. Using this method, what is the time complexity of finding the median of the array? Can you suggest a better method? Can you implement both these methods in Python and compare against numpy.median routine in terms of time?

The method described in the prompt is an inefficient way to find the median. Finding the $k$-th highest element by performing $k$ passes would lead to a very slow algorithm. To find the median of an array of size $n$, you'd need to find the $(\frac{n}{2})$-th element, requiring approximately $\frac{n}{2}$ passes. Each pass takes $O(n)$ time, so the total time complexity would be $O(\frac{n}{2} \times n) = \boldsymbol{O(n^2)}$.

A better method is to sort the array first. Sorting an array of size $n$ typically takes $\boldsymbol{O(n \log n)}$ time. After sorting, the median can be found in $O(1)$ time by simply selecting the middle element. This is much more efficient than the $O(n^2)$ approach.

An even more optimal algorithm for this specific problem (finding the k-th smallest element) is a selection algorithm like Quickselect, which has an average time complexity of $\boldsymbol{O(n)}$. NumPy's median function is highly optimized and likely uses such an algorithm.

# Method 1: Iterative search (O(n^2) approach)
# This is a simplified interpretation of the "n-pass" method
def find_median_slow(arr):
    n = len(arr)
    temp_arr = list(arr)
    for i in range(n // 2 + 1):
        min_val = temp_arr[i]
        min_idx = i
        for j in range(i + 1, n):
            if temp_arr[j] < min_val:
                min_val = temp_arr[j]
                min_idx = j
        temp_arr[i], temp_arr[min_idx] = temp_arr[min_idx], temp_arr[i]
    return temp_arr[n // 2] if n % 2 != 0 else (temp_arr[n // 2 - 1] + temp_arr[n // 2]) / 2


# Method 2: Sorting (O(n log n) approach)
def find_median_sorting(arr):
    sorted_arr = sorted(arr)
    n = len(sorted_arr)
    mid = n // 2
    if n % 2 == 1:
        return sorted_arr[mid]
    else:
        return (sorted_arr[mid - 1] + sorted_arr[mid]) / 2

  1. What is the gradient of the following function with respect to x and y?

$x^2y + y^3sin(x)$

The function is $f(x,y)=x^2y + y^3sin(x)$

The gradient of a function is a vector that contains all of its partial derivatives. For the function $f(x,y)=x^2y + y^3sin(x)$, we need to compute the partial derivatives with respect to x and y. The partial derivative with respect to x is: $$\frac{\partial f}{\partial x} = 2xy + y^3cos(x)$$ The partial derivative with respect to y is: $$\frac{\partial f}{\partial y} = x^2 + 3y^2sin(x)$$

The gradient of f(x,y), denoted as f, is the vector: f(x,y) = $[f/x, f/y] = [2xy + y^3cos(x), x^2 + 3y^2sin(x)]$

def argmin(arr):
    """Compute the index of the minimum element in a numpy array."""
    min_value = arr[0]
    min_index = 0
    for i in range(1,len(arr)):
        if arr[i] < min_value:
            min_value = arr[i]
            min_index = i

    return min_index
